# 🌍 Travel Diary Android App

## 📌 Overview

Travel Diary is an Android mobile application designed for travel enthusiasts to digitally record, manage, and share their travel experiences. The application allows users to create travel memories by adding journal entries, uploading photos, saving trip details, and sharing experiences.

The main aim of this project is to replace traditional travel diaries with a digital platform where users can securely store and revisit their memorable journeys anytime.

---

## ✨ Features

### 🔐 User Authentication
- User registration and login using Firebase Authentication
- Secure user account management

### 📖 Travel Diary Management
- Create travel diary entries
- Add travel title, description, and images
- View and update travel memories
- Delete unwanted entries

### 📸 Image Upload & Storage
- Upload travel photos
- Store images using Firebase Storage
- Display images efficiently using Glide Library

- Add travel location details
- Integration with Google Maps API for geotagging

- Like travel posts
- Add comments

### 👤 User Profile
- Display user profile information
- Manage personal travel memories
